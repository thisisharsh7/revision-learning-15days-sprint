# What to Learn for GSoC Interview

## Technical Skills for Routing QA Project

### 1. Python (Data Generation Script)
- [ ] Basic Python syntax (if/else, loops, functions)
- [ ] Reading/writing JSON files
- [ ] Working with coordinates (lat/lon)
- [ ] Making HTTP requests to Valhalla API
- [ ] Writing scripts that run from command line

### 2. React & TypeScript (Web App)
- [ ] React hooks (useState, useEffect)
- [ ] TypeScript basics (types, interfaces)
- [ ] Component structure
- [ ] State management (already know Zustand)
- [ ] Handling large lists (virtual scrolling)

### 3. MapLibre GL JS (Map Display)
- [ ] Adding layers to map
- [ ] Drawing lines (route geometry)
- [ ] Adding markers (start/end points)
- [ ] Comparing two routes on same map
- [ ] Map events (click, hover)

### 4. GitHub Actions (CI/CD)
- [ ] What is GitHub Actions
- [ ] How to trigger on PR
- [ ] Running scripts in workflow
- [ ] Uploading artifacts
- [ ] Environment variables

### 5. Testing & QA Concepts
- [ ] What is regression testing
- [ ] How to compare results
- [ ] What makes a good test case
- [ ] How to detect route changes
- [ ] Understanding test coverage

### 6. Valhalla Routing Concepts
- [ ] How Valhalla routing works
- [ ] Route request format
- [ ] Route response format
- [ ] What is route geometry
- [ ] Different costing models (car, bike, walk)
- [ ] Turn-by-turn instructions

### 7. OpenStreetMap Data
- [ ] What is OSM
- [ ] Road types (highway, residential, etc)
- [ ] Access restrictions
- [ ] Turn restrictions
- [ ] Why OSM matters for routing

## Interview Topics to Prepare

### About Your Proposal
- [ ] Explain the problem you're solving
- [ ] Walk through your architecture
- [ ] Explain the Python script approach
- [ ] Explain the web app features
- [ ] Justify your timeline

### Technical Questions
- [ ] How will you handle 10,000 routes in browser?
- [ ] How will you detect route differences?
- [ ] What data structure for storing results?
- [ ] How to make comparison fast?
- [ ] How to handle edge cases?

### Your Experience
- [ ] Walk through your 6 PRs
- [ ] Challenges you faced
- [ ] What you learned from C++ contribution
- [ ] Why you want this project
- [ ] Your availability and commitment

### Questions for Mentors
- [ ] Ask about test dataset design
- [ ] Ask about GitHub Actions setup
- [ ] Ask about deployment approach
- [ ] Ask about mentor availability
- [ ] Ask about success metrics
